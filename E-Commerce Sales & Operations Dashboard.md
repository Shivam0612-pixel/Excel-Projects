**Excel Dashboard Project – E-Commerce Sales Analysis**

I recently completed a self-practice project where I built an interactive E-Commerce Sales Dashboard using Microsoft Excel.
This project helped me practice data cleaning, data analysis, and dashboard creation using real-world style datasets.

• KPI Creation
• Dashboard Design
• Data Visualization

Through this project, I improved my understanding of how dashboards can help businesses track sales performance and delivery efficiency.
